# Student-management-system-
-
It helps institutions efficiently handle student data such as admissions, attendance, marks, courses, and fees in a centralized system instead of manual records.

-
 Key Features
Student Registration
Store personal details (name, ID, age, department, contact info)
Course Management
Assign courses/subjects to students
Track enrolled courses
Attendance Management
Daily/subject-wise attendance tracking
Marks & Grade Management
Store exam marks
Generate grades and reports
Fee Management
Track fee payments and dues
User Roles
Admin, Teacher, Student access control
Reports
Student report cards
Attendance and performance summaries
